📐 Bingham-Plastic Equations
Industry-standard rheological model
PV = θ600 − θ300
// Plastic Viscosity (mPa·s)

YP = θ300 − PV
// Yield Point (Pa × 0.4788 → Pa)

τ = YP + PV × γ̇
// Bingham shear stress

Re = 109 × ρ × v × d / PV
// Modified Reynolds number

📐 Volume Equations
Based on SPE annular volumetrics
V_ann = (π/4) × (D²hole − D²casing) × L × (1 + We)
// Annular volume with excess factor

V_casing = (π/4) × D²cid × L_casing
// Casing internal volume (displacement)

V_tail = V_ann × (tail_pct / 100)
// Tail slurry at bottom of annulus

V_lead = V_ann − V_tail
// Lead slurry fills remaining annulus
